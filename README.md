# Resume Builder

Create professional resumes and CVs with customizable templates. Download as PDF or HTML.

## Features

- **3 Templates**: Minimal, Modern, Professional
- **PDF & HTML Export**: Generate downloadable resumes instantly
- **Dark/Light Mode**: Automatic system theme detection
- **Free Tier**: Minimal template available at no cost

## Getting Started

```bash
npm install
npm run dev
```

Visit [http://localhost:5173](http://localhost:5173)

## Build

```bash
npm run build    # Production build
npm run preview  # Preview production build
```

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- jsPDF + html2canvas
